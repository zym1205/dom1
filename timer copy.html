JavaScript定时器入门教程

1. setTimeout - 延迟执行一次

设置一个定时器，在指定的时间后执行一次代码


// 基本语法
setTimeout(函数, 延迟时间);

// 例子
setTimeout(function() {
    console.log("3秒后执行");
}, 3000);

// 取消定时器
let timerId = setTimeout(函数, 延迟时间);
clearTimeout(timerId);

2. setInterval - 重复执行

设置一个定时器，每隔指定的时间重复执行代码


// 基本语法
setInterval(函数, 间隔时间);

// 例子
setInterval(function() {
    console.log("每秒执行一次");
}, 1000);

// 取消定时器
let intervalId = setInterval(函数, 间隔时间);
clearInterval(intervalId);
